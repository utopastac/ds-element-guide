Scrolling lists

Horizontally scrolling list

Shows a list of products/items scrolling horizontally to the right. Only use this when the user doe not need to compare between two items and does not need to have a full understanding of how amny items are in the list - for instance showing product recommendations. Only use when you have 3 or more items in the list.

Metrics
padding top and bottom 16px

Vertically scrolling list

Try to avoid unless absolutely necessary. Full width vertical scrolling components within an already scrollable view are confusing. Try alternate techiniques, such as 'sticky' headers to leave the main view to scroll normally. If absolutely necessary, make sure the data in the list is one dimensional only.

Metrics
padding top and bottom 16px
Border top and bottom 1px solid $grey-light
Inner shadow top and bottom y: -2
blur: 2
colour: rgba(0,0,0,0.12)
